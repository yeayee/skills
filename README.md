

# 极简创业者 —— Claude Code 技能包

基于 Sahil Lavingia 的著作《The Minimalist Entrepreneur》（极简创业者）打造的 Claude Code 技能集。

## 安装方法

在 Claude Code 中输入：

```
/plugin marketplace add slavingia/skills
/plugin install minimalist-entrepreneur
```

即可完成安装 —— Claude Code 会自动从仓库拉取并注册全部 10 个技能。

<details>
<summary>备选方法：从本地克隆安装</summary>

```bash
git clone https://github.com/slavingia/skills.git ~/.claude/plugins/skills
```

然后在 Claude Code 中执行：

```
/plugin marketplace add ~/.claude/plugins/skills
/plugin install minimalist-entrepreneur
```

</details>

## 技能列表

| 技能 | 命令 | 使用场景 |
|------|------|----------|
| **寻找社区** | `/find-community` | 正在寻找商业想法，想找到属于自己的社区 |
| **验证想法** | `/validate-idea` | 测试一个商业想法是否值得投入 |
| **MVP** | `/mvp` | 准备构建第一个产品，但难以控制范围 |
| **流程化** | `/processize` | 有一个产品想法，想先通过手动方式交付价值，再编写代码 |
| **首批客户** | `/first-customers` | 已有产品，需要找到前 100 名客户 |
| **定价** | `/pricing` | 设置价格，或考虑调整价格 |
| **营销计划** | `/marketing-plan` | 已实现产品-市场匹配，准备通过内容进行规模化增长 |
| **可持续增长** | `/grow-sustainably` | 关于支出、招聘或扩张的决策 |
| **公司价值观** | `/company-values` | 定义企业文化，准备开始招聘 |
| **极简审查** | `/minimalist-review` | 对任何商业决策进行直觉式复核 |

## 极简创业者之旅

这些技能按照书中的 progression（发展顺序）设计：

1. **社区** —— 先找到你的用户群体
2. **验证** —— 确认这个问题值得解决
3. **构建** —— 先手动交付，再将其产品化
4. **流程化** —— 把产品想法转化为今天就能手动交付的流程
5. **销售** —— 一个一个地找到前 100 名客户
6. **定价** —— 从第一天起就开始收费
7. **营销** —— 通过内容建立受众
8. **增长** —— 保持盈利，可持续地成长
9. **文化** —— 建造你愿意居住其中的“房子”（企业文化）
10. **审查** —— 将极简原则应用到每一个决策中
